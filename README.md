# Proyecto Final – Página Web Responsive

Sitio web responsive con menú lateral, tabla de datos y diseño adaptable.

## Estructura

```
proyecto-final/
├── index.html
├── productos.html
├── css/
│   └── estilos.css
└── img/
    ├── banner.svg
    ├── producto.svg
    └── nosotros.svg
```

## Tecnologías

- HTML5
- CSS3 (variables, grid, flexbox, media queries)
- JavaScript (menú hamburguesa responsive)

## Páginas

- **index.html** – Página principal con banner, tarjetas de resumen y tabla de inventario.
- **productos.html** – Listado completo de 8 productos con tabla detallada.

## Cómo abrir

Clona el repositorio y abre `index.html` en tu navegador. No requiere servidor ni dependencias.
